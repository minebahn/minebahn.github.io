HOW TO GET A VERSION/PROJECT/FILE ID

Modrinth:
Version ID is located in the "Metadata" section of the VERSION page.

Curseforge PROJECT ID:
Project ID is located in the "About" section of the main project page.

Curseforge FILE ID:
File ID is in the last part of the file URL:
https://www.curseforge.com/minecraft/mc-mods/create/files/4835191
